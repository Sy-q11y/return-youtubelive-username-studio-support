# return-youtube-live-username

YouTubeのライブチャットで、ユーザー名の表示形式を自由に切り替えられるYouTube Studio対応のChrome拡張機能です。
本家を作ったlenlinoさんに感謝します。

# 導入方法

1. このプロジェクトをZip形式でダウンロードする
2. ダウンロードしたZipを解凍する
3. [chrome://extensions/](chrome://extensions/)へ行き、右上の「デベロッパー モード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」を押して先程解凍したフォルダを選択する
